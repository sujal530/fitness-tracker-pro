# Fitness Tracker Pro 💪

This is a simple fitness tracker web application that I built using Python and Flask.  
The main idea of this project is to track daily fitness activities and give basic suggestions.

## Features

- User login and registration system
- Add daily data like steps, calories, and water intake
- View progress and history
- Smart fitness suggestions (based on input)
- Basic AI chat for fitness-related questions

## Tech Used

- Python (Flask)
- SQLite database
- HTML, CSS (for frontend)

## How to Run

1. Install Python on your system  
2. Install Flask using:
   pip install flask  
3. Run the app:
   py app.py  
4. Open browser and go to:
   http://127.0.0.1:5000  

## About Project

I created this project to understand how web applications work using Flask.  
It helped me learn about routing, database handling, and frontend integration.
